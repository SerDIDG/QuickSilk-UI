Areas

Header:

  • main - standard wide area
  • logo - for site logo
  • info - for contact info / etc
  • icons - for social icons
  • links - for login links / secondary menu
  • menu - for primary navigation menu
  • search - for search module
  • langs - for lang switcher module
  • center - for additional modules
  • splash - wide drop zone for image above header

Footer:

  • main - standard wide area
  • copyright - for copyrights and rich editor in it
  • icons - for social icons
  • links - for privacy policy links / secondary menu
  • sitemap - for module menu and sitemap view
  • powered - for QuickSilk CMS copyright
